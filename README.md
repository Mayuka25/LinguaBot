# 🤖 LinguaBot – Multilingual Campus Chatbot

LinguaBot is an AI-powered multilingual campus assistant developed for **Smart India Hackathon (SIH) 2025 – Problem Statement SIH25104**. It helps students access important college information instantly in their preferred language, making campus communication more accessible and efficient.

## ✨ Features

- 🌐 Supports **5 Languages**
  - English
  - Tamil
  - Hindi
  - Telugu
  - Malayalam
- 🤖 AI-powered responses using **LLaMA 3.3 70B** via **Groq API**
- 🔍 Automatic language detection
- 💬 Multi-turn conversation support
- 🎤 Voice input (Speech-to-Text)
- 🔊 Voice output (Text-to-Speech)
- 📄 Downloadable conversation logs
- ❓ FAQ quick-access panel
- 📞 Intelligent fallback with admin contact details for unanswered campus queries
- 📱 Responsive and user-friendly interface

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES2020)
- Groq API
- LLaMA 3.3 70B
- Web Speech API

## 🎯 Problem Statement

Students frequently visit administrative offices for routine queries such as:
- Fee payment
- Scholarships
- Exam schedules
- College timings
- Certificate procedures

LinguaBot provides accurate answers instantly in multiple Indian languages, reducing administrative workload and improving accessibility.

## 🚀 How It Works

1. User enters a question by typing or voice.
2. LinguaBot automatically detects the language.
3. The query is processed using the Groq API with the LLaMA 3.3 70B model.
4. The chatbot responds in the same language.
5. If the query is outside the knowledge base, the chatbot displays the college administration contact information.
